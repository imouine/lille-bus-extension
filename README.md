# Lille Bus Extension

Open-source Chrome extension that displays the next bus arrival time for Lille public transport directly on the extension icon using the **Extension Action Badge**.

The user can choose:
- a line
- a stop
- a direction

The extension then shows the time remaining before the next bus arrives directly on the extension icon.

## Project status

This project is currently in early development.

## Goals

- Simple and fast Chrome extension
- Useful daily transport information
- Clean open-source codebase
- Easy community contributions

## Disclaimer

This project is not affiliated with, endorsed by, or sponsored by any public transport operator in Lille.

It relies on external public transport data sources that may change over time.

## Planned features

- Select a bus line
- Select a stop
- Select a direction
- Show next arrival time in the extension badge
- Change badge color depending on remaining time
- Save user preferences locally
- Refresh data automatically

## Tech stack
